ISpecification<'Actual, 'Model>

A specification for an object under test, based on an abstract model of the object's behavior.

Instance members

Instance memberDescription
GenCommand(arg1)
Signature: 'Model -> Gen<ICommand<'Actual,'Model>>
Modifiers: abstract

Generate a number of possible commands based on the current state of the model. The commands are just hints to speed up execution; preconditions are still checked.
Initial()
Signature: unit -> 'Actual * 'Model
Modifiers: abstract

Initial state of both object and model.
